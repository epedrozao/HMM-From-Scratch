## 概要

このライブラリは、NumPyを使用した純粋なPythonで書かれた、離散観測を持つ**隠れマルコフモデル（HMM）**の完全かつ教育的な実装を提供します。以下の基本アルゴリズムを含みます：

- **Viterbi**（デコーディング）
- **Forward**（評価）
- **Backward**（後向き確率）
- **Forward-Backward**（平滑化）
- **Baum-Welch**（EMによる学習）

各アルゴリズムは2つのバージョンで利用可能です：

- **最適化版**：長い系列でアンダーフローなしに動作するための対数と数値的に安定した手法を使用。
- **理論版**：教科書の数式を忠実に再現しており、アルゴリズムの学習と理解に最適。

このライブラリは教育目的と実際のプロジェクトの両方を想定して設計されており、**自然言語処理**、**定量金融**、**バイオインフォマティクス**、その他の分野における、より専門的なアプリケーションの基盤として機能します。

## クイックスタート

```python
import numpy as np
from hmm import HMM

# シンプルなモデルの定義（2状態、2観測）
A  = np.array([[0.7, 0.3],
               [0.4, 0.6]])
B  = np.array([[0.8, 0.2],
               [0.4, 0.6]])
pi = np.array([0.8, 0.2])

hmm = HMM(2, 2, A=A, B=B, pi=pi)

# 観測系列（0：幸せ、1：不機嫌）
obs = [0, 0, 1, 0]

# Viterbiによるデコーディング
path, prob = hmm.viterbi(obs)
print("最尤状態系列:", path)

# Forwardによる対数尤度
_, log_lik = hmm.forward(obs, return_log=True)
print("対数尤度:", log_lik)

# Forward-Backwardによる平滑化
gamma = hmm.forward_backward(obs)
print("平滑化確率:\n", gamma)

# Baum-Welchによるパラメータ推定（単一系列）
obs_train = [0, 0, 1, 0, 0, 1, 0, 0]
log_lik_final = hmm.baum_welch(obs_train, max_iter=50)
print("学習後の最終対数尤度:", log_lik_final)

# Baum-Welchによるパラメータ推定（複数系列）
obs_multi  = np.array([0, 0, 1, 0, 0, 1, 0, 1, 1, 0])
lengths    = [4, 3, 3]
log_lik_multi = hmm.baum_welch_mult(obs_multi, lengths, max_iter=50)
print("最終対数尤度（複数系列）:", log_lik_multi)
```

## メソッドリファレンス

### コンストラクタ

```python
HMM(n_estados, n_obs, A=None, B=None, pi=None)
```

| パラメータ | 型 | 説明 |
|-----------|------|-------------|
| `n_estados` | `int` | 隠れ状態数 N |
| `n_obs` | `int` | 観測シンボルの種類数 M |
| `A` | `ndarray`、省略可 | 遷移行列、shape (N, N)。各行の和が1。 |
| `B` | `ndarray`、省略可 | 出力確率行列、shape (N, M)。各行の和が1。 |
| `pi` | `ndarray`、省略可 | 初期確率ベクトル、shape (N,)。和が1。 |

指定しない場合、`A`、`B`、`pi`はランダムに初期化され、確率的になるよう正規化されます。

---

### 最適化メソッド（数値的に安定）

| メソッド | 説明 |
|--------|------|
| `viterbi(obs, return_log=False)` | 最尤状態系列（Viterbiデコーディング） |
| `forward(obs, return_log=False)` | 前向き行列 α と全体尤度 P(O\|λ) |
| `backward(obs, return_log=False)` | 後向き行列 β |
| `forward_backward(obs, return_log=False)` | 平滑化状態確率 γ |
| `baum_welch(obs, max_iter, tol, return_log_history)` | EMパラメータ推定 — 単一系列 |
| `baum_welch_mult(obs_concatenated, lengths, max_iter, tol, return_log_history)` | EMパラメータ推定 — 複数系列 |

---

#### `viterbi(obs, return_log=False)`

観測系列に対して最も確率の高い隠れ状態系列を求めます。数値アンダーフローを防ぐため対数確率を使用。

**戻り値：** `path`（int のリスト）、`best_log_prob` または `best_prob`（float）

---

#### `forward(obs, return_log=False)`

前向き行列 α_t(j) = P(o_1, …, o_t, q_t = j | λ) と全体尤度 P(O | λ) を計算します。

**戻り値：**
- `return_log=True` の場合：`log_alpha`（ndarray、shape T×N）、`log_likelihood`（float）
- `return_log=False` の場合：`alpha`（ndarray）、`likelihood`（float）

---

#### `backward(obs, return_log=False)`

後向き行列 β_t(j) = P(o_{t+1}, …, o_T | q_t = j, λ) を計算します。

**戻り値：**
- `return_log=True` の場合：`log_beta`（ndarray、shape T×N）
- `return_log=False` の場合：`beta`（ndarray）

---

#### `forward_backward(obs, return_log=False)`

平滑化確率 γ_t(j) = P(q_t = j | O, λ) を計算します。

**戻り値：**
- `return_log=True` の場合：`log_gamma`（ndarray、shape T×N）
- `return_log=False` の場合：`gamma`（ndarray、各行の和が1）

---

#### `baum_welch(obs, max_iter=100, tol=1e-6, return_log_history=False)`

単一の観測系列からモデルパラメータ（π, A, B）を推定するExpectation-Maximization（EM）アルゴリズム。対数尤度の相対変化が `tol` を下回るか、`max_iter` に達するまで反復します。

**戻り値：**
- `return_log_history=False` の場合：`log_likelihood`（float）
- `return_log_history=True` の場合：`log_likelihood`（float）、`log_lik_history`（float のリスト）

---

#### `baum_welch_mult(obs_concatenated, lengths, max_iter=100, tol=1e-6, return_log_history=False)`

Baum-Welchの複数系列への拡張。複数の独立した観測系列を1つの1次元配列に連結して受け取り、各系列の長さを別途指定します。十分統計量（γ、ξ）はE-stepで全系列にわたって累積され、M-stepでパラメータが一括再推定されます。これは全対数尤度 Σ_s log P(obs_s | モデル) を最大化することと等価です。

| パラメータ | 型 | 説明 |
|-----------|------|-------------|
| `obs_concatenated` | `array-like of int` | 全系列を連結した1次元配列。 |
| `lengths` | `array-like of int` | 各系列の長さ。合計が `len(obs_concatenated)` と一致する必要があります。 |
| `max_iter` | `int` | EMの最大反復回数。 |
| `tol` | `float` | 全対数尤度の相対変化に対する収束許容値。 |
| `return_log_history` | `bool` | `True` の場合、反復ごとの対数尤度リストも返します。 |

**戻り値：**
- `return_log_history=False` の場合：`log_likelihood`（float）— 最終全対数尤度 Σ_s log P(obs_s | モデル)
- `return_log_history=True` の場合：`log_likelihood`（float）、`log_lik_history`（float のリスト）

---

### 理論版メソッド（対数なし、学習用）

これらのメソッドは教科書の数式をそのまま実装しており、アルゴリズムの動作を理解するのに最適です。

> **警告：** これらのバージョンは線形空間で動作するため、長い系列では数値アンダーフローが発生する可能性があります。本番環境では最適化版を使用してください。

| メソッド | 戻り値 |
|--------|---------|
| `viterbi_teorico(obs)` | `path`（int のリスト）、`prob`（float） |
| `forward_teorico(obs)` | `alpha`（ndarray、T×N）、`prob`（float） |
| `backward_teorico(obs)` | `beta`（ndarray、T×N） |
| `forward_backward_teorico(obs)` | `gamma`（ndarray、T×N） |
| `baum_welch_teorico(obs, max_iter, tol, return_log_history)` | `baum_welch` と同じ |
